# RAYCASTER
This project is to learn about the raycasting techniques used in DOOM and Wolfesntein3D.
I am using this project to learn more about the C language as well as create a simple 
raycasting game.

### FEATURES
- [x] 2D MAP
- [x] Player Movement
- [x] 2D Raycast
- [ ] Untextured Walls
- [ ] Textured Walls
- [ ] Camera Mouse Movement
